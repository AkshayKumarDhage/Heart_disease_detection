How this Heart disease detection works?

Heart disease Dataset

Medical report data is collected from various rescources,to form a dataset which includes data of both people suffering from heart disease and those who are healthy.Based on this dataset Machine Learning model is trained.

Machine Learning Model

Using Logistic Regression Algorithm and KNN Algorithm develop a Machine Learning model which is trained on collected medical dataset.

Save the trained ML Model

saving the trained model for future predictions on input values.

Make Predictions

On entering any individual's medical data as input to ML model it predicts the individual as suffering from heart disease or is healthy.
